# python string

## string array example

```python
>>> strValue = "hello world"
>>> strValue
'hello world'
>>> strValue[:4]
'hell'
>>>
>>> strValue[:-4]
'hello w'
>>> len(strValue)
11
>>> strValue[-1:]
'd'
>>> strValue[-2:]
'ld'
>>> strValue[-2:4]
''
>>> strValue[-2:-1]
'l'
>>> strValue[-2:0]
''
>>> strValue[-2:]
'ld'
>>> strValue[-1:]
'd'
>>> strValue[:]
'hello world'
>>> strValue[0:4]
'hell'
```
