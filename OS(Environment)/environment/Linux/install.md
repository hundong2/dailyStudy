# install util

## 01. CMAKE 

[CMAKE HOME PAGE](https://cmake.org/download/)  

download cmake recently version  

```bash
wget https://github.com/Kitware/CMake/releases/download/v3.24.1/cmake-3.24.1.tar.gz
tar -zxvf cmake-3.24.1.tar.gz
cd cmake-3.24.1
./bootstrap
make
sudo make install 
cmake --version 
```
