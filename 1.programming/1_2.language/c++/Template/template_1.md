# C++ Template 1
## definition
```c++
template <typename T> //정의 되는 class에 대해 template을 정의하고, 인자로 T를 받게 되고, T는 반드시 어떠한 타입의 이름을 명시. 
```

## example
```c++
Vector<int> int_vec;
```
의 경우, `<>`안에 `int`라는 Type을 Template에 전달하라는 의미. 
Template을 통해 Type을 전달할 수 있게 되었음.
