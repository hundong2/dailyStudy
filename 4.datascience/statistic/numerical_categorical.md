# numerical data, categorical data ( 수치형, 범주형 데이터)

## 1.numerical

### 1.1 연속형 (continuous data)
- 실제로 관측 가능한 값은 연속적인 척도로 주어진다. 
#### example
- 키, 몸무게

### 1.2 이산형 (descrete data)
- 기록 된 값
#### example
- 주사위 결과, 교통사고 건수

======

## 2. categorical 

### 2.1 순위형 (ordinal data)
- 범주간에 순서의 의미가 있는 자료. 
#### example
- 평점, 선호도

### 2.2 명목형 (nominal data)
- 혈액형과 같이 범주간에 순서의 의미가 없는 자료. 
#### example
- 혈액형, 성별
