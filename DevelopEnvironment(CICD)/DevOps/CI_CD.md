# What is CI/CD?

## CI 
- 지속적 통합(Continuous Integration)
## CD
- 지속적 배포(Continuous Deployment) 

## CI/CD types
- Jenkins  
- CircleCI  
- TravisCI  
- Github Actions  
- etc
